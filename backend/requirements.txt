# combination on Raspberry Pi OS Bookworm. RPi.GPIO is deliberately

# dead end. Both are pure-Python/small-C and available as prebuilt
# armv7 wheels via the piwheels index already configured in the
# Dockerfile, so this doesn't reintroduce a long compile on the Pi.
